📖 Key Formulas & Definitions (Sheet 2)
📤 Transmission Delay
Dtrans = L / R
L = packet size (bits), R = link rate (bps)
⚡ Propagation Delay
Dprop = d / s
d = distance (m), s = propagation speed (2×10⁸ m/s typical)
⏳ Queuing Delay
Increases when traffic intensity (L·a/R) → 1
🖥️ Processing Delay
Time to inspect header, lookup routing, error check.
🔁 Nodal Delay
dnodal = dproc + dqueue + dtrans + dprop
🌍 End-to-End Delay
Sum of all nodal delays along path + store-and-forward contributions.
📦 Store-and-Forward
Router receives full packet before forwarding → adds full transmission delay per hop.
📈 Throughput
Min(R1, R2, ...) bottleneck link.
